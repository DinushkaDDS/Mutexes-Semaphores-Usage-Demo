Run the following lines of code after opening a terminal in the project folder.

1) javac Simulator.java
2) java Simulator

Then input the required number of riders and required number of buses to arrive to the bus stop.
The bus arrivals and departs will be printed on the terminal with the respective amount of riders in it.

If the program hanging out, without any output means that sevaral riders are waiting in the bus stop to bus to arrive.
(Try changing the number of buses to successfully end the program)
